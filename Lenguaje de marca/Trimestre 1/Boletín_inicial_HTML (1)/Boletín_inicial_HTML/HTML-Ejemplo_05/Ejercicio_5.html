La World Wide Web (del ingles, Telaraña Mundial), la Web WWW,
es un sistema de hipertexto que funciona sobre Internet.
Para ver al información se utiliza una aplicación llamada navegador
web para extraer elementos de información (llamados "documentos" o "páginas web")
de lso servidores web (o "sitios") y mostrarlos en la pantalla del usuario,
El usuaio puede entonces seguir hiperenlaces que hay en la pagina
a otros documentos o inclulso enviar información al servidor para interactuar
con él.
